# Dispenser Planting

A NeoForge mod that allows dispensers to automatically plant crops and saplings.

---

## How to make it work

* Put soil block in front of dispenser
* This means dispenser's mouth is obscured by soil
* Put plantable item in dispenser
* Power dispenser

---

## Supported by default

### Crops

* Wheat
* Beetroot
* Carrots
* Potatoes
* Nether Wart
* All saplings
* Nether fungi
* Mushrooms
