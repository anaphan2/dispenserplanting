# Dispenser Planting

A NeoForge mod that allows dispensers to automatically plant crops, flowers, and saplings.

---

## Features

* Dispensers can plant seeds automatically
* Supports vanilla crops (wheat, carrots, potatoes, etc.)
* Supports flowers and saplings
* Supports tall plants (rose bushes, sunflowers, etc.)
* Tag-based system (easy to extend + mod compatible)
* Prevents placing on invalid or blocked blocks

---

## How it works

When a dispenser fires a plantable item:

* It checks the block in front of the dispenser
* Verifies the soil is valid
* Ensures space is available
* Places the correct plant if possible
* Consumes one item from the stack

---

## Supported by default

### Crops

* Wheat
* Beetroot
* Carrots
* Potatoes
* Nether Wart
* Melon Seeds
* Pumpkin Seeds

## Installation

1. Install NeoForge for your Minecraft version
2. Download the latest release `.jar`
3. Place it in your `mods` folder
4. Launch the game
